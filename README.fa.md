# سیستم مدیریت وظایف (کانبان)

یک اپلیکیشن مدیریت وظایف مدرن به سبک کانبان که با **Cloudflare Workers** و پایگاه داده **D1** ساخته شده است.

## ویژگی‌ها

- 📋 **برد کانبان** با سه ستون: انجام نشده، در حال انجام، انجام شده
- 🖱️ پشتیبانی کامل از **Drag & Drop** بین ستون‌ها
- ✏️ افزودن، ویرایش و حذف وظیفه
- 🏷️ اولویت‌بندی وظیفه (کم، متوسط، زیاد) با رنگ‌بندی
- 📅 تاریخ سررسید + نمایش کارهای عقب‌افتاده
- 🔍 جستجو و فیلتر وظایف
- 📊 آمار لحظه‌ای (مجموع، انجام نشده، در حال انجام، انجام شده، سررسید گذشته)
- 🌐 **پشتیبانی دو زبانه** (انگلیسی و فارسی با RTL کامل)
- 🌓 **تم دارک و لایت** با ذخیره‌سازی تنظیمات
- 📱 طراحی واکنش‌گرا (موبایل، تبلت و دسکتاپ)
- 💾 ذخیره‌سازی دائمی داده‌ها در **Cloudflare D1**

## تکنولوژی‌های استفاده شده

- **بک‌اند**: Cloudflare Workers
- **دیتابیس**: Cloudflare D1
- **فرانت‌اند**: HTML + Tailwind CSS + JavaScript
